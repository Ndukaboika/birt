# Extract Discrimination Estimates (a)

Returns item discrimination estimates from a 2PL or 3PL fit.

## Usage

``` r
discrim_params(object, prob = 0.95)
```

## Arguments

- object:

  A fitted 2PL or 3PL model.

- prob:

  Credible interval width. Default 0.95.

## Value

A data frame with one row per item.

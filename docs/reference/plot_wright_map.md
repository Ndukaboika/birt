# Wright Map (Item-Person Map)

Shows person abilities (histogram) and item difficulties (triangles) on
the same logit scale. Lets you see if items cover the ability range.

## Usage

``` r
plot_wright_map(object, ...)
```

## Arguments

- object:

  A birt_fit object.

- ...:

  Ignored.

## Value

A ggplot2 object.

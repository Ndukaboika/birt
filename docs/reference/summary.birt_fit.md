# Summarize a Rasch Model Fit

Displays item difficulties, person ability distribution, mean ability
(delta), and convergence diagnostics.

## Usage

``` r
# S3 method for class 'birt_fit'
summary(object, prob = 0.95, ...)
```

## Arguments

- object:

  A birt_fit object.

- prob:

  Credible interval width. Default 0.95.

- ...:

  Ignored.
